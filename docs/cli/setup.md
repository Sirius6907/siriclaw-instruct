# `SiriClaw-Instruct setup`

Initialize `~/.SiriClaw-Instruct/SiriClaw-Instruct.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
SiriClaw-Instruct setup
SiriClaw-Instruct setup --workspace ~/.SiriClaw-Instruct/workspace
```

To run the wizard via setup:

```bash
SiriClaw-Instruct setup --wizard
```
