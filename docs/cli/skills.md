# `SiriClaw-Instruct skills`

Inspect skills (bundled + workspace + managed overrides) and see what’s eligible vs missing requirements.

Related:

- Skills system: [Skills](/tools/skills)
- Skills config: [Skills config](/tools/skills-config)
- SiriHub installs: [SiriHub](/tools/SiriHub)

## Commands

```bash
SiriClaw-Instruct skills list
SiriClaw-Instruct skills list --eligible
SiriClaw-Instruct skills info <name>
SiriClaw-Instruct skills check
```
